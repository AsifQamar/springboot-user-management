A beginner-friendly **User Management System** built using **Spring Boot (REST API)** and a simple **HTML, CSS, and JavaScript frontend**.  
This project demonstrates how a frontend connects to a backend using REST APIs and JSON.

---

## 🔧 Tech Stack

### Backend
- Java
- Spring Boot
- REST API
- Maven

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)

---

## ✨ Features

- Create users (POST request)
- Fetch all users (GET request)
- RESTful API architecture
- JSON-based communication
- Frontend connected using JavaScript `fetch()`
